3배수 5배수

자연수가 들어있는 리스트에 3의 배수와 5의 배수 중 어떤 수가 더 많은지 구해주세요.

입력

2 3 6 9 12 15 10 20 22 25

출력

리스트에 3의 배수의 개수가 더 많다면 "three"를, 5의 배수의 개수가 더 많다면 "five"를, 3의 배수와 5의 배수의 개수가 같다면 "same"을 return 해주세요.
three

풀이

A = list(map(int,input().split()))
n = len(A)
cnt3 = 0
cnt5 = 0
for i in range(n):
    if A[i] % 3== 0:
        cnt3 += 1
    if A[i] % 5== 0:
        cnt5 += 1

if cnt3 > cnt5:
    print('three')
elif cnt3 < cnt5:
    print('five')
else:
    print('same')
