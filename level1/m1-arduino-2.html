LED analog

LED 불의 밝기가 0.1초마다 점점 밝아지게 한다. 255 밝기가 되면 불을 끈다. 이과정을 반복한다.

입력

없음

출력

없음

풀이

int ledPin = 6;
int bright = 0;
void setup()
{
  pinMode(ledPin, OUTPUT);
}

void loop()
{
  analogWrite(ledPin, bright);
  bright = bright + 10;
  if (bright >= 255) {
    bright = 0;
  }
  delay(100); 
}
