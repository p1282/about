원그리기

거북이가 10개의 원을 점점 커지게 그린다

입력

없음

출력

10개의 크기가 다른 원

풀이


import turtle

turtle.bgcolor('black')
turtle.pencolor('red')
turtle.pensize(4)
turtle.shape('turtle')

n=0
while True:
    n = n + 1
    if n >= 10:
        break
    turtle.penup()
    turtle.goto(0, -n*20)
    turtle.pendown()
    turtle.circle(n * 20)
