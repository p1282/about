길이5이상인 단어

단어들이 들어있는 리스트에서 길이가 5 이상인 단어를 리스트에 들어있는 순서대로 이어 붙이려 합니다. 예를 들어 리스트가 다음과 같은 경우 ["my", "favorite", "color", "is", "violet"] 라면 "favoritecolorviolet"을 만들면 됩니다.

입력

my favorite color is violet

출력

favoritecolorviolet

풀이

A = list(input().split())
n = len(A)
B = []

for i in range(n):
    if len(A[i]) >= 5:
        B.append(A[i])

print(B)
