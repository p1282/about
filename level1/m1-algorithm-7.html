n-queen

n크기의 ​정사각형 모양의 보드판에서 n개의 queen이 서로 공격하지 못하게 놓을 수 있는 방법의 수를 구하시오. queen은 8방향으로 공격할 수 있다. 가로, 세로, 위, 아래, 오른쪽 증가, 오른쪽 감소, 왼쪽 증가, 왼쪽 감소 이며 한번 움직이는 크기는 보드판 끝까지 가능하다. 즉 한 행에 하나의 queen 만 놓을 수 있다. 또한 한 열에도 하나의 queen 만 놓아야만 서로 공격하지 못한다.

입력

체스판의 크기가 숫자로 입력된다

4
-------
8

출력

2
-------
92

풀이


n = int(input())
Q = [0] * n

def promising(k):
    i = 0
    while i < k:
        if Q[i] == Q[k] or \
           Q[i] - Q[k] == (i-k)  or \
           Q[i] - Q[k] == (k-i):
            return False
        i += 1
    return True

ans = 0
def nQueens(k):
    global ans

    if k == n:
        ans += 1
        return

    i = 0
    while i < n:
        Q[k] = i
        if promising(k):
            nQueens(k+1)
        i += 1

nQueens(0)
print(ans)
