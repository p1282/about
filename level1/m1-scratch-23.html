짝수의 제곱합

서로 다른 두 자연수 n과 m이 매개변수로 주어질 때, n부터 m까지의 자연수 중에서 짝수들의 제곱의 합을 구하시오.

입력

두 자연수 n과 m이 입력됩니다.
4 7

출력

4부터 7까지의 자연수 중에서 짝수는 4와 6입니다. 짝수의 제곱합은 52 입니다. 4^2 + 6^2 = 16 + 36 = 52
52

풀이

n,m = map(int,input().split())
t = 0
for i in range(n,m+1,1):
    if i % 2==0:
        t += (i ** 2)

print(t)
