주사위 2개값

2개의 주사위를 던졌을때 나오는 값을 보여준다.

입력

없음

출력

임의로 던져진 주사위 값이 2개 출력된다

풀이

import random

while True:
    print('주사위 2개를 굴립니다....')
    print('주사위는 다음과 같습니다')
    a = random.randint(1,6)
    b = random.randint(1,6)
    
    print(a, b)
    print()
    ans = input('계속 할래요?[y/n]')
    if ans == 'n' or ans == 'N':
        break
