중복문자삭제

알파벳 문자열이 주어질 때, 연속하는 중복 문자를 삭제하려고 합니다.
예를 들어, "senteeeencccccceeee"라는 문자열이 주어진다면, "sentence"라는 결과물이 나옵니다.
연속하는 중복 문자들을 삭제한 결과를 구해주세요.

입력

senteeeencccccceeee

출력

sentence

풀이

s = input()
n = len(s)
pre = s[0]
A = [pre]
for i in range(1,n):
    if s[i] == pre:
        continue
    else:
        A.append(s[i])
    pre = s[i]

for i in range(len(A)):
    print(A[i],end='')
