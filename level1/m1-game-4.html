태극무늬 그리기

거북이가 태극무늬를 그린다

입력

없음

출력

태극무늬

풀이


import turtle

a = 88
turtle.shape('turtle')
turtle.bgcolor('black')
turtle.pensize(2)
turtle.speed(0)

for i in range(300):
    if i % 2 == 0:
        turtle.color('red')
    else:
        turtle.color('blue')
    turtle.forward(i)
    turtle.left(a)
