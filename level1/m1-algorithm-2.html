3의 배수게임

3의 배수 게임을 하던 정올이는 잦은 실수로 벌칙을 받게 되었다.
정올이를 도와주기 위해 3의 배수 게임에서 왕이되기 위한 프로그램을 작성하시오.

입력

하나의 정수가 입력된다
7

출력

1부터 입력된 수까지 출력하는 데, 단 3의 배수이면 숫자 대신에 X를 출력한다.
1 2 X 4 5 X 7

풀이

n = int(input())
for i in range(1,n+1,1):
    if i % 3 == 0:
        print('X', end=' ')
    else:
        print(i, end=' ')
