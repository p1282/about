인기투표

N명의 후보에 대해 투표한 결과가 들어있는 리스트가 있습니다. 예를 들어 5명의 후보에 대해 투표를 진행한 결과가 [2, 5, 3, 4, 1, 5, 1, 5, 5, 3]이라면 순서대로 [2번, 5번, 3번, 4번, 1번, 5번, 1번, 5번, 5번, 3번] 후보에 투표했음을 나타냅니다. 이때, 정확히 K 표를 받은 후보는 총 몇 명인지 구하려 합니다. 예를 들어 K = 2일 때, 위 투표 결과에서 정확히 2표를 받은 후보는 1번, 3번 후보로, 총 2명입니다.

입력

인기투표결과리스트
총학생수, K값
2 5 3 4 1 5 1 5 5 3
5 2

출력

5

풀이

A = list(map(int,input().split()))
m,k = map(int,input().split())

B = [0] * (m+1)

for i in range(len(A)):
    for j in range(1,m+1,1):
        if A[i] == j:
            B[j] += 1

for i in range(1,m+1,1):
    if B[i] == k:
        print(i)

#-------------

A = list(map(int,input().split()))
m,k = map(int,input().split())

B = [0] * (m+1)

for i in range(len(A)):
    B[A[i]] += 1

c = 0
for i in range(1,m+1,1):
    if B[i] == k:
        c += 1
print(c)
