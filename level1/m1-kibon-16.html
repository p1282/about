소수갯수

자연수의 각 자릿수 중에서 소수는 몇 개인지 구하려 합니다. 즉, 자연수를 각 자릿수별로 나누었을 때, 2, 3, 5, 7이 몇 개 있는지 구하려 합니다. 예를 들어, 자연수가 29022531일 때, 각 자릿수 중 소수의 위치는 다음과 같습니다. 2 9 0 2 2 5 3 1 따라서 소수는 2 2 2 5 3 으로 총5개 입니다.

입력

29022531

출력

5

풀이

n = int(input())
A = [2,3,5,7]
c = 0

while True:
    m = n % 10
    if m in A:
        c += 1
    n = n // 10
    if n == 0:
        break

print(c)
