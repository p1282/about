수 뒤집기

시작 날짜와 끝 날짜가 주어질 때, 두 날짜가 며칠만큼 떨어져 있는지를 구하려 합니다,
각 달의 날짜 수는 1월부터 순서대로 [31, 28, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31] 이며, 윤년은 고려하지 않습니다.

입력

1423

출력

3241

풀이

A = []
n  = int(input())
while True:
    m = n % 10
    A.append(m)
    n = n // 10
    if n == 0:
        break

print(A)
