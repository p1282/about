linear structure search

오름차순으로 정렬되어 숫자가 입력된다. 여기에서 찾고자 하는 숫자가 입력되면, 해당 숫자의 위치를 알려준다. 만약 입력된 숫자가 없다면 -1을 출력한다.

입력

정렬된 숫자 배열과, 찾고자 하는 숫자
1 2 3 5 7 9 11 15
11

출력

7

풀이

A = list(map(int,input().split()))
k = int(input())
n = len(A)
ans = -1
for i in range(n):
    if A[i]==k:
        ans = i+1
print(ans)
