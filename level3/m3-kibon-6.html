암호화

다음과 같은 규칙으로 문자열을 암호화 하려고 합니다.
- 같은 문마자 나오는 만큼 숫자를 셉니다
- 해당 문자와 숫자를 순서대로 출력합니다
예를 들어 aaaabbb를 암호화 한다고 하면 a4b3이 됩니다. 이처럼 문자열을 입력하면 규칙에 따라 암호화 하는 프로그램을 만드세요.

입력

알파벳 소문자로 이루어진 문자열이 주어집니다. 문자열의 길이는 1보다 크고 100보다 작습니다
aaaabbbc

출력

a4b3c1

풀이

S = list(input())
n = len(S)

pre = S[0]
cnt = 1
for i in range(1,n):
    if S[i] == pre:
        cnt += 1
    else:
        A.append(pre)
        A.append(cnt)
        pre = S[i]
        cnt = 1
A.append(pre)
A.append(cnt)
print(A)
