완전수

완전수란 자기 자신을 제외한 약수들의 합이 자기 자신이 되는 수를 말합니다. 예를 들어 6의 약수는 1,2,3,6 이며 여기서 자기자신 6을 제외한 1,2,3의 합은 6이 됩니다. 조건에 부합해 6은 완전수 입니다. 수를 하나 입력하면 그 수가 완전수인지 확인하는 프로그램을 작성하세요.

입력

1보다 크고 10000보다 작은 자연수 1개가 주어집니다.
6

출력

입력한 수가 완전수라면 o, 아니면 x 가 출력됩니다.
o

풀이

n = int(input())
tot = 0
for i in range(1,n,1):
    if n % i == 0:
        tot = tot + i
if tot == n:
    print('o')
else:
    print('x')
