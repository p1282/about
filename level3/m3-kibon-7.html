복호화

다음과 같은 규칙으로 암호화 된 문자열을 복호화 하려고 합니다. 암호화 규칙은 아래와 같습니다.
- 같은 문마자 나오는 만큼 숫자를 셉니다
- 해당 문자와 숫자를 순서대로 출력합니다
예를 들어 문자열이 a4b3이 들어오면 다시 복호화 하면 aaaabbb 가 됩니다. 이처럼 암호화된 문자열을 입력되면 규칙에 의거해 원래 문장을 다시 복호화 해서 출력하는 프로그램을 만드세요.

입력

a4b3

출력

aaaabbb

풀이

S = list(input())
n = len(S)

cnt = 0
t =  0

pre = S[0]
for i in range(1,n):
    if S[i].isdigit():
        t = (t*10) + int(S[i]) 
    else:
        print(pre * t, ,end='')
        pre = S[i]
        t = 0
print(pre * t)
