윤년 판별

윤년이란 2월달이 28일까지 있습니다. 평연은 2월달이 29일까지 있습니다. 윤년은 다음과 같은 규칙에 따라 정해집니다.
(1) 4로 나누어 떨어지는 해는 윤년으로 합니다.(1988년, 1992년, 1996,,,2028년...) (2) 4, 100으로도 나누어 떨어지는 해는 평년으로 합니다.(1900년, 2100년, 2200년,...)
(3) 4, 100, 400으로 나누어 떨어지는 해는 윤년으로 합니다.(2000녀, 2400년 ...)
연도가 입력되면 윤년인지 평년인지 판별하는 프로그램을 작성하세요.

입력

년도가 주어집니다.
1520

출력

윤년이면 o, 평년이면 x를 출력합니다.
o

풀이

y = int(input())
if y % == 4:
    if y % 100 == 0:
        if y % 400 == 0:
            print('o')
        else:
            print('x')    
    else:
        print('o')	

else:
    print('x')
