단어 총개수

영어 단어들이 하나 이상의 공백을 사이에 두고 늘어서 있습니다. 이 때 영어 단어들이 총 몇 개가 있는지 계산해서 출력하는 프로그램을 만드세요.

입력

영문 소문자로 이루어진 단어들이 하나 이상의 공백을 사이에 두고 문자열 한 줄에 이어서 주어집니다. 단어는 최소 1개 이상이며 문자열 가장 앞에는 공백이 들어올 수 없습니다.
apple banana orange

출력

단어의 총 개수가 출력됩니다.
3

풀이

S = list(input().split())
n = len(S)
cnt = 0
space = False
for i in range(n):
    if S[i] == ' ':
        if  space == False:
            cnt += 1
            space = True
        else:
            continue
    else:
        space = False
        
print(cnt + 1)
