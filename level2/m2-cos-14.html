체력시험

체력시험 합격 인원을 알아보려고 합니다. 체력시험 종목으로는 윗몸일으키기, 팔굽혀펴기, 달리기가 있습니다. 종목별 합격기준은 다음과 같습니다.

| 윗몸일으키기 | 팔굽혀펴기 | 달리기 |
| 80점 이상 | 88점 이상 | 70점 이상 |

* 통과한 종목이 하나 이하거나, 통과 점수의 반을 넘기지 못한 종목이 있다면 불합격입니다.
* 그 외에는 합격입니다.
시험에 합격한 인원을 출력합니다.

입력

학생수가 먼저 입력됩니다. 다음줄부터는 각 학생의 윗몸일으키기, 팔굽혀펴기, 달리기 점수가 차례로 들어옵니다.

2
30 40 100
97 88 95
-------
6
90 88 70
85 90 90
100 100 70
30 90 80
40 10 20
83 88 80

출력

시험합격인원

1
------
4

풀이


n = int(input())

def passcnt(a,b,c):
    x = 0

    if a >= 80:
        x += 1
    if b >= 88:
        x += 1
    if c >= 70:
        x += 1

    return x

def midalcnt(a,b,c):
    y = 0
    if a <= 40:
        y += 1
    if b <= 44:
        y += 1
    if c <= 35:
        y += 1
    return y

ans = 0
for i in range(n):
    a,b,c = map(int,input().split())
    x = passcnt(a,b,c)
    y = midalcnt(a,b,c)
    
    if x <= 1 or y > 0:
        pass
    else:
        ans = ans + 1

print(ans)
