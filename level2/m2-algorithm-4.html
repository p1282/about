minimum sum

n*n개의 수가 주어진다. (1<=n<=10) 이때 겹치지 않는 각 열과 각 행에서 수를 하나씩 뽑는다. (즉, 총 n개의 수를 뽑을 것이다, 그리고 각 수는 100 이하의 값이다.) 이 n개의 수의 합을 구할 때 최솟값을 구하시오.

입력

첫 줄에 n이 입력된다. 다음 줄부터 n+1줄까지 n개씩의 정수가 입력된다.
3
1 5 3
2 4 7
5 3 5

출력

구한 최소 합을 출력한다.
8

풀이


INF=int(1e9)

n = int(input())
A=[]
for i in range(n):
    a = list(map(int,input().split()))
    A.append(a)
col_visited=[False] *n

ans = INF
def fun(row, sumv):
    print(sumv)
    global ans
    if row == n:
        ans =min(ans, sumv)
        return
    for i in range(n):
        if not col_visited[i]:
            col_visited[i]=True
            fun(row+1, sumv+A[row][i])
            col_visited[i]=False

fun(0,0)

print(ans)
