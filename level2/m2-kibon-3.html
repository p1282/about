합격 불합격

시험을 치른 학생들에게 합격, 불합격 정보를 알려줍니다. (75 점 이상 합격) 첫줄에 시험을 치른 학생수가 입력되고, 그 다음 학생들의 점수가 차례로 입력되면, 합격,불합격을 말해줍니다

입력

학생수, 각 학생들의 성적

출력

모든 학생들의 합격, 불합격 여부를 마지막에 출력

풀이

n =  int(input())
A = []
for i in range(n):
    score = int(input())
    if score >= 75:
        A.append('합격')
    else:
        A.append('불합격')
print(A)
