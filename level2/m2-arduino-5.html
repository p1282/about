원하는LED켜기(red,green,blue)

원하는 색상의 LED를 골라서 켠다

입력

색상입력 (red, green, blue)

출력

없음

풀이

빨간색 LED 7번핀에 연결한다.
초록색 LED 8번핀에 연결한다.
파란색 LED 9번핀에 연결한다.
String m = "What color turn on?";
void setup()
{
  Serial.begin(9600);
  pinMode(7, OUTPUT);
  pinMode(8, OUTPUT);
  pinMode(9, OUTPUT);
}

void loop()
{
  Serial.println(m);
  String s = Serial.readStringUntil('\n');
  if (s == "red") {
      digitalWrite(7, HIGH);
      digitalWrite(8, LOW);
      digitalWrite(9, LOW);
  } else if (s == "green") {
      digitalWrite(7, LOW);
      digitalWrite(8, HIGH);
      digitalWrite(9, LOW);
  } else if (s == "blue") {
      digitalWrite(7, LOW);
      digitalWrite(8, LOW);
      digitalWrite(9, HIGH);
  }
  delay(1000); 
}
