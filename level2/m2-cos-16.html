차량2부제

관공서 주차장에서는 차량 2부제를 실시합니다. 차량 2부제는 차량 번호 끝자리가 홀수인 차량은 홀수 일에만, 짝수인 차량은 짝수 일에만 주차장에 들어올 수 있도록 하는 제도입니다. 며칠인지를 나타내는 숫자와 그날 주차장에 들어오려고 하는 차들의 번호들이 입력됩니다. 이 차들에서 주차장에 들어올 수 있는 차량의 수를 구하도록 작성해주세요.

입력

첫째줄에는 며칠인지 날짜가 입력됩니다
둘째줄에는 들어오려고 하는 차들의 번호가 여러개 입력됩니다.

17
3285 1724 4438 2988 3131 2998

출력

17일은 홀수 일이므로 홀수 번호 차량인 1724, 3131 총 2대의 차량만 주차장에 들어올 수 있습니다.

2

풀이

day = int(input())
A = map(int,input().split())
pivot = day % 2
c = 0
for i in range(len(A)):
    x = A[i] % 2
    if x == pivot:
        c += 1
print(c)
