쇠막대기

여러개의 쇠막대기를 레이저로 절단하려고 한다. 효율적인 작업을 위해서 쇠막대기를 아래에서 위로 겹쳐 놓고, 레이저를 위에서 수직으로 발사하여 쇠막대기를 자른다. 쇠막대기와 레이저의 배치는 다음 조건을 만족한다.

- 쇠막대기는 자신보다 긴 쇠막대기 위에만 놓일 수 있다.
- 쇠막대기를 다른 쇠막대기 위에 놓는 경우 완전히 포함되도록 놓으면, 끝점은 겹치지 않도록 놓는다.
- 각 쇠막대기를 자르는 레이저는 적어도 하나 존재한다.
- 레이저는 어떤 쇠막대기의 양 끝점과도 겹치지 않는다.

그림 예제 : 수평으로 그려진 실선은 쇠막대기이고, O은 레이저의 위치, 수직으로 레이저는 발사 된다.
( ( () () ) ( () ) () ) ) ( () )

      O O       O      O        O
    ----------  ------
  --------------------------
-------------------------------  -------

이러한 레이저와 쇠막대기 배치는 괄호로 표현할 수 있다.
() 모든 레이저를 표시한다
( 쇠막대기의 왼쪽끝
) 쇠막대기의 오른쪽끝

위의 예제는 쇠막대기가 레이저 O 에 의해서 몇 개의 조각으로 잘려진다. 가장 위에 있는 두개의 쇠막대기는 각가 3개와 2개의 조각으로 잘려지고, 중간 쇠막대기는 5개로 잘려지고, 가장 아래 쇠막대기는 총 7조각이 난다. 그래서 총 17개의 조각으로 잘려진다. 쇠막대기와 레이저의 배치를 나타내는 괄호 표현이 주어졌을 때, 잘려진 쇠막대기 조각의 총 개수를 구하시오.

입력

쇠막대기와 레이저의 배치가 괄호 표현으로 나타낸다. 괄호문자 개수는 최대 100,000이다.
((((()())(())()))(())
-------
(((()(()()))(())()))(()())

출력

잘린 총 조각 수
17
-------
24

풀이

A = list(input())
n = len(A)

stick = 0
ans = 0

for i in range(n):
    if A[i] == '(':
        stick += 1
    elif A[i] == ')':
        stick -= 1
        if prev == '(':
            ans += stick
        else:
            ans += 1


    prev = A[i]
print(ans)
