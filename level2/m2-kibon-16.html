거북이 도형그리기

입력한 값으로 도형을 그리기

입력

8

출력

8각형

풀이

하나의 도형만 그린다.

import turtle

n = int(input())

turtle.speed(1)
turtle.pensize(3)
turtle.color('blue')

for i in range(n):
    turtle.forward(100)
    turtle.right(360/n)

import turtle

n = int(input())

turtle.speed(0)
turtle.pensize(3)
turtle.color('red')
size = 0
for j in range(10):
    size = size + 10
    for i in range(n):
        turtle.forward(10 + size)
        turtle.right(360/n)

10개의 도형을 크기가 점점 커지고 색상이 변하면서 그린다
import turtle

n = int(input())

turtle.speed(0)
turtle.pensize(3)
C = ['red','blue']

size = 0
for j in range(10):
    turtle.color(C[j%2])
    size = size + 10
    for i in range(n):
        turtle.forward(10 + size)
        turtle.right(360/n)
