초음파센서 HC-SR04

초음파 센서가 다가오는 앞의 물체와의 거리를 계산한다. 만약 거리가 가까와 지면 LED불이 들어온다. 만약 거리가 다시 멀어지면 LED불이 꺼진다.

입력

없음

출력

없음

풀이

LED 는 8번 핀에 연결하고, 
초음파센서는 출력센서는 2번핀에 입력센서는 5번핀에 연결한다.

void setup()
{
  pinMode(2, OUTPUT);
  pinMode(5, INPUT);
  pinMode(8, OUTPUT);
}

void checkdist() 
{
  digitalWrite(2,HIGH);
  delayMicroseconds(10);
  digitalWrite(2,LOW);
  int d = pulseIn(5,HIGH);
  if (d < 200) {
    return true;
  } else {
    return false;
  }
}

void loop()
{
  bool a = checkdist();
  if (a == true) {
    digitalWrite(8,HIGH);
  } else {
    digitalWrite(8,LOW);
  }
  delay(100);
}
