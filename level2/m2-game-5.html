거북이 원그리기

임의의 위치에 임의의 색으로 임의의 크기 원을 그린다 위 작업을 함수를 이용해서 100번 반복한다.

입력

없음

출력

100개의 원

풀이



import turtle, random

turtle.shape('turtle')
turtle.speed(0)
def fun():
    r = random.random()
    g = random.random()
    b = random.random()
    turtle.color(r,g,b)
    x = random.randint(-200,200)
    y = random.randint(-200,200)
    turtle.penup()
    turtle.goto(x,y)
    s = random.randint(10,80)
    turtle.pendown()
    turtle.circle(s)

for i in range(50):
    fun()
