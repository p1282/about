거북이 패턴그리기2

원을 반복적으로 그린다. 크기를 줄이고, 색상을 변화시키면서 반복해서 그린다.

입력

숫자 (원의 갯수를 숫자로 입력)

출력

반복적인 원 그리기

풀이


import turtle

turtle.bgcolor('black')
turtle.shape('turtle')
turtle.speed(0)

b = 0
n = int(input())
s = 180
for j in range(90):
    for i in range(n):
        turtle.color(1,0,b)
        turtle.circle(s)
        turtle.right(360/n)
    b = b + (1/90)
    s = s - (2)
