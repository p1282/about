타일색칠하기

타일을 'R', 'G', 'B' 색으로 칠하려 합니다. R 색으로는 3칸을 한 번에, G 색으로는 2칸을 한 번에 칠할 수 있으며, B 색으로는 1칸을 칠할 수 있습니다. 색은 R, G, B 순서로 한 번씩 번갈아 가면서 사용해야 하며, 타일의 길이를 넘겨서 칠할 수는 없습니다. 예를 들어, 타일 길이가 11이면 "RRRGGBRRRGG"의 색으로 칠할 수 있습니다. 타일 길이가 입력될때 타일을 색칠한 순서를 문자열로 출력하세요. 순서에 맞게 타일을 칠할 수 없다면 -1을 return 해주세요.

입력

타일길이가 입력됩니다. 1000 이하의 자연수 입니다.

11
------------
16

출력

타일을 색칠한 순서를 문자열로 출력 해주세요.

RRRGGBRRRGG
------------
-1

풀이


n = int(input())
if n % 6 == 0:
    m = n // 6    
    ans = 'RRRGGB' * m
    print(ans)
elif n % 6 == 5:
    m =  n // 6
    ans = 'RRRGGB' * m
    ans = ans + 'RRRGG'
    print(ans)
elif n % 6 == 3:
    m = n // 6
    ans = 'RRRGGB' * m
    ans = ans + 'RRR'
    print(ans)
else:
    print(-1)
