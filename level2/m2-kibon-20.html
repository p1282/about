오타 갯수

영어 단어를 외우기 위해 단어를 반복하여 타이핑합니다. 그런데, 타이핑하고 보니 오타가 많습니다. 오타를 수정하려면 문자를 몇 개 바꿔야 하는지 구하려 합니다.

입력

타이핑한 문자들이 입력됩니다.
정확한 문자가 입력됩니다.

CODE COED CDEO
CODE

출력

바꿔야 하는 문자 개수를 출력한다


5

풀이

A = list(input().split())
word = input()

cnt = 0
for i in range(len(A)):
    s = A[i]
    for j in range(len(s)):
        if s[j] == word[j]:
            continue
        else:
            cnt += 1

print(cnt)
