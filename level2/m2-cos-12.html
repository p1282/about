주스만들기

주스 1잔을 만들려면 사과 3개와 당근 1개가 필요합니다. 그런데 키우는 토끼에게 먹이를 주기 위해 사과와 당근 종류에 상관없이 k개를 빼놓으려고 합니다. 주스는 최대한 많이 만들수록 좋습니다. 사과 개수와 당근 개수, 토끼에게 줄 먹이 개수 k가 주어질 때 주스를 최대 몇 잔 만들 수 있는지 구하세요.

입력

사과 개수와 당근 개수, 토끼에게 줄 먹이 개수

5 1 2
------------
10 5 4

출력

최대 주스를 몇잔 만들 수 있는지 출력합니다.

1
------------
2

풀이

a,b,k = map(int,input().split())

if a == b*3:
    km = k//4
    kn = k%4
    if kn == 0:
        print(b - km)
    else:
        print(b - km - 1)

elif a > b*3:
    k = k - (a - (b*3))
    if k > 0:
        km = k //4
        kn = k % 4
        if kn == 0:
            print(b - km)
        else:
            print(b - km-1)
    elif k <= 0:
        print(b)

elif a < b*3:
    k = k - (b - (a//3))
    if k > 0:
        km = k // 4
        kn = k % 4
        if kn ==0:
            print(a//3 - km)
        else:
            print(a//3 - km-1)
            
    elif k <= 0:
        print(a//3)
