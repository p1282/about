거북이 패턴그리기

원하는 도형을 원하는 횟수만큼 반복해서 그린다. 위 작업은 도형을 점점 키우면서 반복한다.

입력

몇각형 그릴까요?
몇개를 그릴까요?
위 작업을 몇번 반복할까요?

출력

8
16
10

풀이



import turtle,random

x = int(input('몇각형 그릴까요'))
y = int(input('몇개를 그릴까요'))
z = int(input('위 작업을 몇번 반복할까요'))

turtle.bgcolor('black')
turtle.shape('turtle')
turtle.speed(0)
turtle.pensize(3)

for k in range(z):
    r = 1
    b = 0
    for j in range(y):
        turtle.color(r,0,b)
        for i in range(x):
            turtle.forward(50+(k*z))
            turtle.right(360/x)
        turtle.right(360/y)
        b = b + (1/y)
